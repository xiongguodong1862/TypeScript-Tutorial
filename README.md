- 克隆项目
```
git clone https://github.com/dogXgod/TypeScript-Tutorial.git
```

- 进入react+redux+ts的demo，ts-project
```
cd ts-project
```

- 下载依赖
```
yarn
```

- 启动项目
```
yarn start
```
